TramRoutes - програма, призначена для допомоги користувачам у виборі трамваїв та складання ними маршруту у місті Львів.

Роботу виконали:
 - Паламар Юлія ПМі-46
 - Кітраль Сергій ПМі-44

Розподіл завдань:
1. збір даних про маршрути усіх трамваїв у Львові та оформлення їх у json форматі - Паламар Юлія
2. написання логіки програми - Кітраль Сергій
3. додавання UI частини програми - Паламар Юлія
4. тестування програми і оформлення відповідного протоколу - Кітраль Сергій
5. написання інструкції для користувача - Паламар Юлія

Ось додатково посилання на гітхаб репозиторій: https://github.com/yuliiapalamar/pythonLab3

Вхідний формат даних:
JSON формат для зупинок трамваїв виглядає наступним чином:
{
  "трамвай1": {
    "forward": ["зупинка1", "зупинка2", "зупинка3"],
    "backward": ["зупинка3", "зупинка2", "зупинка1"]
  },
  "трамвай2": {
    "forward": ["зупинкаA", "зупинкаB", "зупинкаC"],
    "backward": ["зупинкаC", "зупинкаB", "зупинкаA"]
  }
}
трамвай1 та трамвай2 - це ідентифікатори трамваїв.
forward - масив зупинок у напрямку до кінцевої зупинки.
backward - масив зупинок у зворотному напрямку.

Доступні функції програми:
 - Пошук трамваїв за зупинкою: дозволяє знайти трамваї, які проходять через зазначену зупинку.
 - Кількість зупинок і пересадок: надає інформацію про кількість зупинок та необхідних пересадок для обраного маршруту.
 - Трамваї через кілька зупинок: дозволяє знайти трамваї, які зупиняються на кількох вказаних зупинках.
 - Скласти маршрут: допомагає побудувати маршрут з початкової до кінцевої зупинки.

Як користуватися програмою:
1. Для початку роботи потрібно запустити файл main.py.
2. Після запуску програми з'явиться вікно з заголовком TramsRoutes.
3. Оберіть необхідний функціонал, натиснувши на відповідну кнопку:
    - Пошук трамваїв за зупинкою
    - Кількість зупинок і пересадок
    - Трамваї через кілька зупинок
    - Скласти маршрут
4. У відповідному полі введіть назви зупинок, з якими хочете працювати.
5. Натисніть кнопку Пошук, щоб отримати результати.
6. Щоб повернутися на головне вікно натисніть кнопку Назад.
7. Для виходу з програми натисніть кнопку Вийти.
